# Sistemi Genit Cloud — Faza 2

Ky branch shton motorin online për regjistrat, peshimin, blerjet, shitjet dhe stokun.

## Modulet

- Artikuj, kategori, klientë dhe furnitorë
- Formulari i Peshave me zbritje % dhe konfirmim në stok
- Kërkesa për Ofertë
- Porosi Blerjeje
- Pranime Blerjeje
- Fatura Blerjeje
- Oferta Shitjeje
- Porosi Shitjeje
- Fletë-Dalje
- Fatura Shitjeje
- Gjendja e Stokut
- Import nga të dhënat legacy

## Siguria dhe integriteti

- Izolim sipas tenant dhe kompani
- Kontroll aksesesh sipas roleve
- Transaksione PostgreSQL
- Audit Log
- Bllokim i daljes pa gjendje
- Dokumente me status DRAFT / CONFIRMED / CANCELLED
