# Smoke tests — Faza 2

1. Krijo kategori dhe artikull me koeficient koli/paletë.
2. Krijo furnitor dhe klient.
3. Regjistro formular peshe dhe konfirmoje; stoku duhet të rritet.
4. Krijo pranim blerjeje dhe konfirmoje; stoku duhet të rritet.
5. Krijo fletë-dalje me sasi të disponueshme; stoku duhet të ulet.
6. Provo fletë-dalje mbi gjendjen; API duhet ta bllokojë.
7. Verifiko Audit Log për krijim dhe konfirmim dokumentesh.
8. Verifiko izolimin midis kompanive dhe magazinave.
